Tetrad 4.2 Download Site

Common Download Problems and Solutions

  1. When you click "LAUNCH TETRAD," you are asked to download a .jnlp file.
    • Problem 1: Java Web Start was never installed. If Java Web Start was never installed, your machine doesn't know what program to use to open .jnlp files, so it has to ask. The solution is to install JRE 1.4.0 on your machine, which will install Java Web Start for you. Use this link:

      http://java.sun.com/j2se/1.4/download.html

      Previous versions of the JRE did not automatically install Java Web Start; you can install it yourself using this link:

      Download Java Web Start

    However, the Tetrad 4.2 software requires JRE 1.4.0 or higher to be installed anyway, so it makes more sense to just install JRE 1.4.0.

    • Problem #2: You do have Java Web Start installed, but you've still being asked to download .jnlp files. If you actually downloaded the .jnlp file and are trying to double click it to get it to run, you are running into the same problem; Windows still doesn't know what program to use to open it. The solution, on a Windows machine, is to set the "open" action for type "jnlp" to use "javaws.exe" to open files of this type. The easy way to do this is as follows.

      1. Download the .jnlp file. 
      2. Right click the .jnlp file..
      3. Select "Open with..."-->"Choose Program."
      4. Click "Other..." 
      5. Navigate to C:\Program Files\Java Web Start\javaws.exe and open it.
      6. Check "Always use this program to open these file."
      7. Click "OK."

  2. Java Web Start told you that you need version 1.4+ of the JRE, and you dutifully downloaded (and installed!) version 1.4+ of the JRE, but you still can't launch Tetrad.
    • Most likely you didn't register version the version of the JRE you just installed with Java Web Start. You can do this as follows.

      1. Open the Java Web Start console.
        • Windows: If you have the icon still on your desktop or in your Start menu, double click it. Otherwise, find "javaws.exe" and double click that.
        • Linux: Find "javaws" and run it from the command line.
      2. Select "Preferences" and click the "Java" tab.
      3. Add your newly installed JRE 1.4 to the list of JRE's using the tools provided.
