Tetrad 4.2 Download Site

You may launch the most recent version of the Tetrad 4.2 application from this page. For more information about Tetrad project or previous versions of Tetrad, see the Tetrad Project Page.

Software Requirements

You must have Jave Runtime Environment 1.4.1 or higher installed on your computer. (The full J2SDK 1.4.1 will of course do as well.) Instructions for downloading and installing JRE 1.4.1 are on this page:

JRE 1.4.1 Download

Note that Tetrad 4.2 uses Java Web Start to launch. If you install JRE 1.4.1 on a Windows machine, Web Start will be automatically installed for you. On a Linux machine, Web Start will not be automatically installed, but the install files will be included, and there will be a REAME file to explain how to install it.

Apologies to Mac users--Web Start works just fine on a Mac and is automatically installed; however, the latest version of Java available for Mac is still slightly behind the version being used for Tetrad 4.2. Hopefully this situation will be rectified soon (by Apple).

Java Web Start Download

If you have JRE 1.4.1 installed correctly, with Web Start (see above), you can launch Tetrad 4.2 by clicking this link:

Launch Tetrad

If you want to launch Tetrad 4.2 again, you can come back to this link and click it again, or you can launch through the Java Web Start console, or if you prefer, you can use Java Web Start to make shortcuts on your desktop to launch Tetrad 4.2. Whichever method you use, if you are connected to the net, the latest version of Tetrad 4.2 will be downloaded and launched for you automatically.
